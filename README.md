#  Balloons & Snowflakes

An interactive web application featuring floating balloons and snowflake animations, built using Google AI Studio and deployed on Vercel.

## 🌐 Live Demo

https://ballons-and-snow-flakes.vercel.app/

## 🚀 Features

* Interactive visual animations
* Responsive web interface
* Fast deployment using Vercel
* Built as a hands-on learning project

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Vite
* Git & GitHub
* Vercel
